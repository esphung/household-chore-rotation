# Household Chore Rotation

A SwiftUI iOS app for managing and rotating household chores.

## Requirements

* macOS with Xcode 15 or newer
* iOS Simulator runtime installed (for example, iOS 17+)

## Run the App in Xcode

1. Open the project:
   - Double-click `Household Chore Rotation.xcodeproj`

   - Or run:

```bash
     open "Household Chore Rotation.xcodeproj"
```

2. In Xcode, select the **Household Chore Rotation** scheme.
3. Choose an iOS Simulator device (for example, iPhone 15).
4. Press **Run** (or use `Cmd+R`).

## Troubleshooting

* If build errors mention missing simulator runtimes, install one from Xcode settings.
* If Xcode says the selected destination is unavailable, choose a different installed simulator.
